COMANDOS

/lag | Abre a lista de jogadores online e o LAG de cada um



REQUISITOS

- Minetest versão 0.4.14 ou superior
- O jogo precisa estar compilado com RUN_IN_PLACE

INSTALAR

Defina a permissao 775 para o arquivo pingar.sh que está no mod
Comando usado seria:

$ chmod 775 pingar.sh
